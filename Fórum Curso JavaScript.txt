Resposta do moderador para configuração do VSCode com o Live Server instalado nos dois apps:

Ola, pode sim, caso ja tenha instalado e esteja no chrome, siga essas instruções:

Para mudar o navegador padrão que o Live Server utiliza, você pode seguir os seguintes passos:

Abra o Visual Studio Code
Clique em “File” (Arquivo) no menu superior
Selecione “Preferences” (Preferências)
Clique em “Settings” (Configurações)
Na barra de busca, digite “liveServer.settings”
Encontre a opção “Live Server > Settings:CustomBrowser” e escolha o browser que deseja.
Feito isso, o Live Server abrirá o navegador que você definiu como padrão.
